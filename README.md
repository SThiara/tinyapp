## Dependencies

- Node.js
- Express
- EJS
- bcrypt
- body-parser
- cookie-session
- cookie-parser
- method-override

## Getting Started

- Install all dependencies (using the `npm install` command).
- Run the development web server using the `node Tinyapp_server.js` command.
